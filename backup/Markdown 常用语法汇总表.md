
# Markdown 常用语法汇总表

| 效果 | 语法写法 | 说明 |
| :--- | :--- | :--- |
| # 一级标题 | `# 一级标题` | 最大标题 |
| ## 二级标题 | `## 二级标题` | 常用小标题 |
| ### 三级标题 | `### 三级标题` | 小节标题 |
| **粗体** | `**粗体文字**` | 强调重点 |
| *斜体* | `*斜体文字*` | 轻微强调 |
| ~~删除线~~ | `~~删除线~~` | 表示废弃/修改 |
| `行内代码` | `` `代码` `` | 行内小代码 |
| 代码块 | ` ```语言 ` <br> `代码内容` <br> ` ``` ` | 多行代码块 |
| 1. 有序列表 | `1. 第一项` | 自动编号 |
| - 无序列表 | `- 列表项` | 圆点列表 |
| - [ ] 任务 | `- [ ] 未完成` | 待办清单 |
| > 引用 | `> 引用内容` | 引用/备注 |
| 分割线 | `` `---` `` 或 `` `***` `` | 分隔内容 |
| [链接](地址) | `[显示文字](链接地址)` | 普通超链接 |
| ![图片](地址) | `![图片说明](图片URL)` | 插入图片 |
| 表格 | `` `\| 表头 \|` `` <br> `` `\| --- \|` `` <br> `` `\| 内容 \|` `` | 标准表格 |
